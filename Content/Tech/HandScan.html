POSE MEASUREMENTS

For a new application of tailored hand braces, MIGN needed a method for quickly gathering key measurements in the field, and assigning the appropriate brace to the patient. I built a solution based on applying pose detection models, clustering techniques and point cloud manipulation to LiDAR scans collected from an iPhone.






After building a prototype iPhone application to gather point clouds, we created an algorithm for collecting measurements that involved projecting the point cloud into 2-D space to overlay a pose estimation model, then mapping it back to 3D space.




We used the placed markers as reference points to generate cut planes at key joints along each finger.




Once points at the cut plane were collected the table was removed using linear regression, point cloud outliers were removed, the fingers were isolated using clustering, and moving "average" curve was fit to each finger to simplify its profile.
